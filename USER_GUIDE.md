# 🛒 장바구니 사용 가이드

## ⚠️ 중요: 장바구니 추가 방법

### ❌ 작동하지 않는 방법
자연어로 말하기:
- "장바구니에 담아줘"
- "이 향수 장바구니에 넣어줘"
- "구매하고 싶어"

→ **챗봇이 "담았습니다"라고 말하지만, 실제로는 추가되지 않습니다!**

### ✅ 올바른 방법
**향수 카드의 버튼 클릭**:

1. 챗봇에서 "향수 추천해주세요" 입력
2. 향수 카드가 표시됨
3. 카드 하단의 **장바구니 버튼 (🛒)** 클릭
4. "장바구니에 추가되었습니다" 메시지 확인

## 📝 사용 시나리오

### 시나리오 1: 향수 추천 받고 장바구니에 추가

1. **챗봇 열기**
   - 우측 하단 챗봇 아이콘 클릭

2. **향수 추천 요청**
   ```
   사용자: 향수 추천해주세요
   ```

3. **향수 카드 확인**
   - 5개의 향수 카드가 표시됨
   - 각 카드에 브랜드, 이름, 가격 표시

4. **장바구니에 추가**
   - 원하는 향수 카드의 **🛒 버튼** 클릭
   - "✅ [상품명]이(가) 장바구니에 추가되었습니다!" 메시지 확인

5. **장바구니 확인**
   - "장바구니 보기 🛒" 버튼 클릭
   - 또는 우측 상단 장바구니 아이콘 클릭

### 시나리오 2: 여러 향수 추가

1. 첫 번째 향수 카드의 🛒 버튼 클릭
2. "쇼핑 계속하기" 버튼 클릭
3. 다른 향수 추천 받기
4. 두 번째 향수 카드의 🛒 버튼 클릭
5. "장바구니 보기" 버튼으로 확인

## 🎯 버튼 위치

### 향수 카드 버튼
```
┌─────────────────────┐
│   🌸 (이미지)       │
│                     │
│ 브랜드명            │
│ 상품명              │
│ 가격                │
│                     │
│ [상세보기] [🛒] [⚡]│ ← 여기!
└─────────────────────┘
```

- **상세보기**: 상품 상세 정보 모달
- **🛒**: 장바구니에 추가
- **⚡**: 바로 구매

### Quick Action 버튼
챗봇 메시지 하단에 표시되는 버튼들:
- **장바구니 보기 🛒**: 장바구니 내용 확인
- **더 많은 향수 보기**: 추가 추천
- **선호도 변경**: 취향 변경

## 🐛 문제 해결

### 문제: "장바구니가 비어있습니다" 메시지
**원인**: 자연어로 요청했거나, 버튼 클릭이 실패함

**해결**:
1. 향수 카드의 🛒 버튼을 **직접 클릭**
2. 브라우저 콘솔(F12) 확인:
   ```
   🛒 Adding to cart: perfume_xxx
   ✅ Cart updated, items: 1
   ```
3. 로그가 나오지 않으면 페이지 새로고침

### 문제: 버튼 클릭해도 반응 없음
**해결**:
1. 페이지 새로고침 (Ctrl+Shift+R)
2. 브라우저 캐시 클리어
3. 챗봇 닫고 다시 열기

### 문제: "상품 정보를 찾을 수 없습니다"
**원인**: 메시지 히스토리에서 상품을 찾지 못함

**해결**:
1. 챗봇에서 다시 "향수 추천해주세요" 입력
2. 새로 표시된 카드의 버튼 클릭

## 💡 팁

### 1. 향수 카드가 표시되지 않으면
- "향수 추천해주세요" 정확히 입력
- 또는 "플로럴 향수 추천해주세요" 등 구체적으로 요청

### 2. 장바구니 확인 방법
- 챗봇: "장바구니 보기 🛒" 버튼
- 메인 페이지: 우측 상단 장바구니 아이콘

### 3. 빠른 구매
- 향수 카드의 **⚡ 버튼** 클릭
- 장바구니를 거치지 않고 바로 결제

## 📱 모바일에서 사용

1. 챗봇 아이콘 탭
2. 향수 추천 요청
3. 카드 스크롤
4. 🛒 버튼 탭
5. 장바구니 확인

## 🎉 성공 예시

```
사용자: 향수 추천해주세요

챗봇: [5개 향수 카드 표시]

사용자: [첫 번째 카드의 🛒 버튼 클릭]

챗봇: ✅ 조르지오 아르마니 - 씨 오 드 퍼퓸이(가) 
      장바구니에 추가되었습니다!
      
      현재 장바구니: 1개 상품, 총 0원
      
      [장바구니 보기] [쇼핑 계속하기]

사용자: [장바구니 보기 클릭]

챗봇: 🛒 장바구니 (1개 상품, 총 0원)
      
      [상품 카드 표시]
      
      [결제하기] [장바구니 비우기] [쇼핑 계속하기]
```
